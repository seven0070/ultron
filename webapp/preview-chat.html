Explain how black holes work in one paragraph a 10-year-old could understand.
🧠
Monad

Imagine you drop a marble on a stretchy trampoline. A tiny marble makes a little dip. Now imagine dropping a bowling ball — it makes a huge dip, so deep that anything rolling nearby falls in.

A black hole is like that bowling ball, but for space itself. It's a place where gravity is so strong that even light — the fastest thing in the universe — can't crawl back out once it gets too close. That's why we can't see a black hole directly; we only see how it bends the light around it, like a magnifying glass in space.

I'll create monad/plugins/auto_timezone.py with a TimeZonePlugin that returns local time, UTC time, and the detected zone using Python's stdlib zoneinfo — no external deps.

pythonfrom datetime import datetime
from zoneinfo import ZoneInfo
from monad.plugins.manager import Plugin

class TimeZonePlugin(Plugin):
    id = "time_zone"
    name = "Time & Zone"
    description = "Reports local + UTC time and detected zone."

    def on_load(self):  pass
    def on_unload(self): pass

    def now(self):
        return {
            "local":     datetime.now().isoformat(),
            "utc":       datetime.utcnow().isoformat() + "Z",
            "zone_name": str(datetime.now().astimezone().tzinfo),
        }
